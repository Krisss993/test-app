Here are similar Algorithms for Both Regression and Classification:

Decision Trees:

  • Regression: Decision trees predict continuous values at the leaves by averaging the target values.
  • Classification: Decision trees assign categorical labels to data points by traversing through the splits.

Random Forest:

  • Regression: Random forests average the predictions of multiple decision trees to provide a continuous value.
  • Classification: Random forests use a majority vote among trees to assign a class label.

K-Nearest Neighbors (KNN):

  • Regression: KNN regression predicts a value by averaging the target values of the k-nearest neighbors.
  • Classification: KNN classification assigns a label based on the majority class among the k-nearest neighbors.

AdaBoost:

  • Regression: AdaBoost for regression boosts weak learners to reduce prediction errors.
  • Classification: AdaBoost for classification focuses on reducing classification errors by adjusting sample weights after each iteration.

AdaBoost Regressor

Description: AdaBoost combines weak learners (typically decision trees) to create a strong model, adjusting weights for instances based on the error of previous models. Hyperparameters:

  • n_estimators: Number of boosting stages (trees) to be run.
  • learning_rate: Shrinks the contribution of each tree to control overfitting.

Decision Tree Regressor

Description: Decision tree regressor splits the data recursively based on feature values to create a tree structure that predicts the target variable by averaging outcomes in the leaf nodes.

Hyperparameters:

  • criterion: The function to measure the quality of a split (squared_error, absolute_error, poisson).
  • max_depth: Maximum depth of the tree.
  • min_samples_split: Minimum number of samples required to split a node.
  • min_samples_leaf: Minimum number of samples required in a leaf node.
  • max_features: The number of features to consider when looking for the best split.

Random Forest Regressor

Description: Random forest is an ensemble of decision trees, where each tree is trained on a bootstrap sample of the data, and predictions are averaged to improve accuracy and reduce overfitting.

Hyperparameters:

  • n_estimators: Number of trees in the forest.
  • max_depth: Maximum depth of each tree.
  • min_samples_split: Minimum number of samples required to split an internal node.
  • min_samples_leaf: Minimum number of samples required in a leaf node.
  • max_features: The number of features to consider when looking for the best split.
  • bootstrap: Whether to use bootstrap samples when building trees.

K-Nearest Neighbors (KNN) Regressor

Description: KNN regressor predicts the target value for a data point by averaging the values of its k-nearest neighbors in the feature space.

Hyperparameters:

  • n_neighbors: Number of neighbors to consider.
  • weights: Weight function to apply to neighbors (uniform or distance).
  • algorithm: Algorithm to compute nearest neighbors (auto, ball_tree, kd_tree, brute).
  • eaf_size: Leaf size for the tree-based algorithms.
  • p: The power parameter for the Minkowski distance metric (1 for Manhattan distance, 2 for Euclidean distance).
