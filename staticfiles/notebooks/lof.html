LOF Scores:¶

The negative outlier factor scores are extracted from the LOF model. These scores represent how "outlying" each point is. Higher negative values indicate more outlier-like behavior.

Radius Calculation:¶

A radius is calculated based on the LOF scores, which is used to visually scale the points in the next plot (larger circles for outliers).

In [35]:
LOF_scores = lof.negative_outlier_factor_
radius = (LOF_scores.max() - LOF_scores) / (LOF_scores.max() - LOF_scores.min())

Here, the outliers are visualized by plotting larger circles around the outliers. The circle size is proportional to the radius calculated from the LOF scores. This helps in understanding the degree to which a point is considered an outlier.
