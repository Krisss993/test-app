Lasso Regression

Lasso regression is a regularized linear regression that adds an L1 penalty to reduce overfitting by shrinking less important coefficients to zero.

dataset = pd.read_csv('https://storage.googleapis.com/esmartdata-courses-files/ml-course/insurance.csv')
df = dataset.copy()

for col in df.columns:
    if df[col].dtype == 'object':
        df[col] = df[col].astype('category')

df[df.duplicated()]
df.drop_duplicates(inplace=True)
df = pd.get_dummies(df, drop_first=True, dtype='int')

X = df.copy()
y = X.pop('charges')

scaler = StandardScaler()
X = scaler.fit_transform(X)
X_train, X_test, y_train, y_test = train_test_split(X,y,test_size=0.2)

Hyperparameters:

  • alpha: Controls the strength of regularization (higher values mean more regularization).
  • fit_intercept: Whether to include an intercept in the model.
  • max_iter: Maximum number of iterations allowed for the optimization algorithm.

# Lasso regressor
lasso = Lasso()

params = {
    'alpha': [0.1, 1.0, 10.0],
    'fit_intercept': [True, False],  
    'max_iter': [1000, 5000, 10000] 
}

grid_search = GridSearchCV(lasso, params, cv=5, scoring='r2')

grid_search.fit(X_train, y_train)

best_params = grid_search.best_params_
best_model = grid_search.best_estimator_

print("Best Parameters:", best_params)
print("Best Model:", best_model)
 
r2 = best_model.score(X_test, y_test)
print("Lasso R2:", r2)

y_pred = best_model.predict(X_test)
