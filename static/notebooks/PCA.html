The features sepal_length, petal_length, and petal_width are selected for PCA analysis. Before performing PCA, the features are standardized using StandardScaler() to ensure that they have a mean of 0 and standard deviation of 1.

The matrix measures the relationship between the variables (sepal length, petal length, and petal width) after standardization.

Using np.linalg.eig(), the eigenvalues and eigenvectors of the covariance matrix are calculated. These values are essential for PCA:

  • Eigenvalues tell how much variance each principal component explains.
  • Eigenvectors indicate the directions of the principal components in feature space.

The explained variance ratio is computed by dividing each eigenvalue by the total sum of eigenvalues. This metric shows how much variance each principal component captures.

A 2-component PCA is performed by selecting the two eigenvectors that correspond to the largest eigenvalues. These vectors are stacked to form a matrix W, which is then used to project the original data (X_std) into a 2D space:
